What does it do?

Detect the a person's face from the provided image and estimate the age and gender. The face detection functionality was developed using OpenCV library and the classification part, using Keras library

Steps

Access the Calculation page, upload an image that contains a face and press the calculate button. The result should appear on the right side

Technology

The Agen application is developed using Microservices architecture and deployed using Docker.

Results

The best accuracy for gender classification, during validation, is approximately 79,1%. The best accuracy for age classification, during validation, is approximately 24,2%.
